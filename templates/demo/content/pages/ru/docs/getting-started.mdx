# Быстрый старт

## 1. Создаем проект

```bash
npm create astro@latest -- --template affiliatefm/website-core-template
cd my-site
make install && make bootstrap && make seed
```

`make bootstrap` → создаёт `affiliatefm.core.json` с перечнем защищенных путей. `make seed` → заполняет `src/data/` демо-метриками.

## 2. Основные команды

| Команда | Что делает |
| --- | --- |
| `make dev` | dev-сервер Astro |
| `make build` | production-билд + форматирование HTML |
| `make preview` | предпросмотр билда |
| `make check` | типизация/MDX проверки |
| `make seed` | перегенерирует sample data |
| `make update` | вытягивает свежий core без затрагивания protected paths |

## 3. Контент и локали

- Английские страницы → `src/content/pages/*.mdx`
- Русские страницы → `src/content/pages/ru/...`
- Для разных URL используйте `permalink` + `alternates` в frontmatter.

## 4. Данные

- Кладите JSON в `src/data/`.
- Импортируйте через `@/lib/data` → доступны хелперы для витрин.
- Демонстрационный компонент `<DataShowcase />` можно встраивать прямо в MDX.

## 5. Обновления

- `make update` скачивает репозиторий `affiliatefm/website-core-template` во временную папку.
- Копируются только файлы из `corePaths`, остальные пути из `protectedPaths` пропускаются.
- После копирования проверьте diff, выполните `npm install` (если `package.json` изменился) и соберите проект заново.

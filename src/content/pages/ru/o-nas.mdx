# Об этом шаблоне

Этот бойлерплейт демонстрирует как построить **content-driven многоязычный сайт** с i18n возможностями Astro 5.

## Архитектура

### Конфигурация (`src/config/site.ts`)

**Единое место** для всех настроек сайта:

- Поддерживаемые локали и дефолтная локаль
- URL сайта и метаданные
- UI переводы и ссылки навигации

### Контент (`src/content/pages/`)

MDX файлы по локалям. Файлы дефолтной локали в корне, остальные — в подпапках.

### Роутинг (`src/pages/`)

Единый универсальный роут:

- `[...slug].astro` — Все локали, все пути

## Кастомные URL

Эта страница демонстрирует **кастомный URL**:

- English: `/about`
- Русский: `/ru/o-nas`

Достигается через `alternates` во frontmatter обеих страниц + `permalink` на русской версии.

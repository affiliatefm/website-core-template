# Зачем нам это ядро

Affiliate.FM публикует независимые исследования и инструменты для партнерского и performance-маркетинга. Нам нужен был статичный стек, который легко переиспользовать, обновлять и передавать командам в разных странах.

## Принципы

1. **Только статический вывод** — Astro 5 (static) → любой CDN/хостинг.
2. **Жесткие конвенции** — контент в `src/content/pages/<locale>/`, UI в `src/components/` и layouts в `src/layouts/`.
3. **Документируем решения** — `make bootstrap` создаёт `affiliatefm.core.json`, где фиксируются защищенные пути и версия ядра.
4. **Performance + a11y** — базовые темы уже настроены, можно сосредоточиться на контенте.

## Рабочий процесс

1. `npm create astro@latest -- --template affiliatefm/website-core-template`
2. `make bootstrap && make seed`
3. Пишем контент и добавляем локали (достаточно создать папку `src/content/pages/ru/...`).
4. Используем `make update`, чтобы подтянуть свежий core без затирания `protectedPaths`.

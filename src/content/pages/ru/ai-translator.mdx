## Установка

```bash
npm install @affiliate.fm/astro-content-ai-translator
npx astro-ai-translator init
```

Мастер установки выполнит:
- Добавление интеграции в ваш `astro.config.mjs`
- Настройку вашего OpenAI API-ключа в `.env`

## Использование

### Отметьте файлы для перевода

В ваших контент-файлах добавьте `_translateTo` во фронтматтер:

```yaml
---
title: О нас
description: Узнайте о нашей компании
_translateTo: [ru, de]
---
```

Варианты:
- `_translateTo: [ru, de]` — перевод на указанные языки
- `_translateTo: all` — перевод на все настроенные языки
- Без поля — файл не переводится

### Запустите перевод

```bash
npx astro-ai-translator
```

Перед началом перевода вы увидите:
- **Оценку стоимости** — приблизительное количество токенов и стоимость в USD
- **Список файлов** — что будет переведено
- **Запрос подтверждения** — одобрите перед тратой кредитов API

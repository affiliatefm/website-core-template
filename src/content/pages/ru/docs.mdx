## Структура проекта

```
src/
├── content/pages/     # ✏️ Ваш контент (MDX-файлы)
│   ├── index.mdx      #    Главная страница
│   ├── docs.mdx       #    Документация
│   ├── _collections/  #    Элементы коллекций (страницы продуктов)
│   │   ├── software/
│   │   │   └── ai-translator.mdx
│   │   └── websites/
│   │       └── ssg-directory.mdx
│   ├── software/      #    Лэндинг коллекции
│   │   └── index.mdx
│   └── websites/      #    Лэндинг коллекции
│       └── index.mdx
│   └── ru/            #    Контент на русском
│       └── index.mdx
├── data/              # ✏️ Ваша конфигурация
│   ├── site.ts        #    URL сайта, локали, шаблоны, коллекции
│   └── ui.ts          #    Переводы интерфейса
├── config/            #    Конфигурация рантайма (автогенерируется)
├── i18n/              #    Хелперы для URL
├── integrations/      #    Проверки сборки
├── templates/           #    Макеты страниц
└── pages/             #    Роутер Astro
```

**Ваши файлы** (✏️) — редактируйте свободно, не затрагиваются при обновлениях  
**Ядро** — обновляется через `make update`

## Конфигурация

### 1. Настройка сайта

Отредактируйте `src/data/site.ts`:

```typescript
export const siteUrl = "https://your-domain.com";
export const locales = ["en", "ru"] as const;
export const defaultLocale = "en";
```

### 2. Добавьте строки интерфейса

Отредактируйте `src/data/ui.ts`:

```typescript
export const uiStrings = {
  en: { siteName: "My Site" },
  ru: { siteName: "Мой сайт" },
} as const;
```

## Добавление контента

Создайте MDX-файлы в `src/content/pages/`:

```yaml
---
title: "Page Title"
description: "Мета-описание для SEO"
---

# Ваш контент здесь

Пишите на Markdown с полной поддержкой компонентов MDX.
```

### Опции frontmatter

| Поле | Тип | Описание |
|------|-----|----------|
| title | string | Заголовок страницы (обязательно) |
| description | string | Мета-описание |
| updatedAt | date | Для sitemap |
| draft | boolean | Исключить из сборки |
| link | string | Внешний URL (страницы продуктов) |
| pros | string[] | Список плюсов (страницы продуктов) |
| cons | string[] | Список минусов (страницы продуктов) |

## Многоязычная настройка

### Одинаковые слаги (автоматически)

```
pages/about.mdx      → /about/
pages/ru/about.mdx   → /ru/about/
```

Hreflang-ссылки генерируются автоматически.

## Расширение с помощью инструментов

Шаблон отлично работает с другими инструментами Affiliate.FM:

<RatingTable collection="software" limit={2} offset={1} />

### AI-перевод

Добавьте автоматический перевод в ваш рабочий процесс:

```bash
npm install @affiliate.fm/astro-content-ai-translator
npx astro-ai-translator init
```

Затем отметьте файлы для перевода:

```yaml
---
title: About Us
_translateTo: all
---
```

### Добавление интеграций

```bash
npx astro add tailwind
npx astro add react
```

## Обновление

Шаблон спроектирован для безопасных обновлений:

```bash
# Посмотреть, что изменится
make update-check

# Применить обновление
make update
```

**Сохраняется:** `src/content/`, `src/data/`, `public/`  
**Обновляется:** всё остальное

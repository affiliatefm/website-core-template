# Только на русском

Эта страница существует **только на русском**.

## Что происходит

Когда у страницы нет перевода:

- Переключатель показывает другие локали как недоступные
- `hreflang` теги не генерируются для отсутствующих локалей
- Страница доступна только по своему URL

## Когда использовать

Страницы одной локали подходят для:

- Регионального контента
- Юридических страниц, специфичных для страны
- Временного контента

Обратите внимание — "English" показан как недоступный.

# Affiliate.FM · Открытое ядро для performance-команд

Мы делимся инструментами для партнерского маркетинга. Этот репозиторий можно взять за основу, кастомизировать и регулярно обновлять без боли.

## Что внутри

- **Контент** — MDX + Astro content collections, строгие схемы, автоссылки на переводы.
- **Обновления** — `make bootstrap` фиксирует защищенные пути, `make update` подтягивает свежий core.
- **Данные** — `make seed` генерирует demo JSON в `src/data/`, вы можете заменить их загрузками из своего BI.
- **Инструменты** — Makefile + npm scripts + интеграция `@affiliate.fm/astro-content-ai-translator`.

<DataShowcase />

## Как работать

1. `make install && make bootstrap && make seed`
2. Добавьте контент в `src/content/pages/ru/...`
3. Настройте навигацию и домены в `src/config/site.ts`
4. Обновляйтесь через `make update`

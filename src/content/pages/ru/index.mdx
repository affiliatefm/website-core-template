# Astro i18n Бойлерплейт

Чистая основа для создания многоязычных сайтов на **Astro 5**.

## Возможности

- **Content-driven** — Страницы как MDX файлы
- **Автосвязывание** — Переводы связываются по структуре
- **Кастомные URL** — Разные слаги для разных локалей
- **Типизация** — Полная поддержка TypeScript
- **Минимализм** — Без лишних зависимостей

## Демо страницы

- [Главная](/ru) — Связана с `/` (авто)
- [О нас](/ru/o-nas) — Кастомный URL вместо `/ru/about`
- [Документация](/ru/docs/getting-started) — Вложенный путь
- [Только RU](/ru/only-russian) — Нет английской версии

## Быстрый старт

```bash
npm install
npm run dev
```

## Как это работает

Контент в `src/content/pages/`. Структура папок определяет URL:

```
src/content/pages/
├── index.mdx           → /
├── about.mdx           → /about
└── ru/
    ├── index.mdx       → /ru
    └── o-nas.mdx       → /ru/o-nas
```

Страницы с одинаковой структурой связываются автоматически. Кастомные URL — через `permalink` + `alternates`.

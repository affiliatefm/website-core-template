# Базовый шаблон сайта

Минималистичный шаблон Astro 5 для многоязычных статических сайтов. Не требует настройки — просто добавьте контент.

Создан [Affiliate.FM](https://affiliate.fm) — независимые медиа и open-source инструменты для аффилиатного, перфоманс- и диджитал-маркетинга.

## Быстрый старт

```bash
npm create astro@latest -- --template affiliatefm/website-core-template
cd my-site
make dev
```

Откройте [http://localhost:4321](http://localhost:4321)

## Возможности

- **Без настройки** — Работает сразу «из коробки»
- **Многоязычный** — Встроенная i18n с поддержкой hreflang
- **Безопасен для обновлений** — Обновления ядра не затрагивают ваш контент
- **Ориентирован на контент** — MDX с полной поддержкой компонентов
- **Проверки сборки** — Автоматическая валидация при каждой сборке

## Обзор

AI-ассистент, который преобразует черновики Markdown в хорошо структурированные страницы. Добавляет правильную иерархию заголовков, блоки-вызовы, таблицы и обогащает метаданные — при этом сохраняет ваш стиль и содержание.

## Что он делает

Преобразует это:

```markdown
---
title: My Post
---

here is some text about topic. it has important info.
also there are steps: first do this, then that, finally done.
```

В это:

```markdown
---
title: My Post
description: Подробное руководство по теме с пошаговыми инструкциями.
---

# My Post

Здесь представлен текст о **теме**. Он содержит важную информацию.

## Шаги

1. Сначала сделайте это
2. Затем сделайте то  
3. Наконец, всё готово

> **Примечание:** Это руководство предполагает базовое знакомство с темой.
```

## Возможности

- **Улучшение структуры** — Добавляет правильную иерархию заголовков
- **Насыщенные блоки** — Преобразует шаблоны в блоки-вызовы, таблицы, блоки кода
- **Генерация метаданных** — Создаёт описания, теги на основе контента
- **Сохранение стиля** — Сохраняет ваш авторский стиль
- **Выборочная обработка** — Отмечайте файлы с помощью `_enhance: true`

## Установка

```bash
npm install @affiliate.fm/astro-content-ai-enhancer
```

## Использование

Отметьте файлы для улучшения:

```yaml
---
title: My Draft
_enhance: true
---
```

Запустите улучшатель:

```bash
npx astro-ai-enhancer
```

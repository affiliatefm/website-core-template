## Обзор

Утилита для перевода на базе искусственного интеллекта для сайтов на Astro. Автоматически генерирует локализованный контент из файлов в директории `content/` и поддерживает как одностраничные, так и многостраничные i18n-процессы.

## Установка

```bash
npm install @affiliate.fm/astro-content-ai-translator
npx astro-ai-translator init
```

Мастер настройки выполнит:
- Добавление интеграции в ваш `astro.config.mjs`
- Настройку вашего OpenAI API-ключа в `.env`

## Использование

### Отметьте файлы для перевода

В ваших файлах контента добавьте `_translateTo` во frontmatter:

```yaml
---
title: О нас
description: Узнайте о нашей компании
_translateTo: [ru, de]
---
```

Варианты:
- `_translateTo: [ru, de]` — перевод на выбранные языки
- `_translateTo: all` — перевод на все настроенные языки
- Без поля — файл не переводится

### Запуск перевода

```bash
npx astro-ai-translator
```

Перед началом перевода вы увидите:
- **Оценку стоимости** — приблизительное количество токенов и стоимость в USD
- **Список файлов** — что будет переведено
- **Запрос подтверждения** — подтвердите перед расходом средств API

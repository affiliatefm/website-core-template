## Обзор

Полный каталог статических генераторов сайтов. Сравнивайте функции, фильтруйте по языкам и находите идеальный SSG для вашего следующего проекта.

## О проекте

staticsitegenerators.net — это реальный пример того, что можно создать с помощью website-core-template. Здесь представлены:

- **Мультиязычный контент** — Один и тот же контент на разных языках с корректным hreflang
- **Страницы на основе данных** — Списки SSG, сгенерированные из структурированных данных
- **SEO-оптимизация** — Корректные мета-теги, карта сайта, структурированные данные
- **Автоматическое развертывание** — GitHub Actions → GitHub Pages

## Возможности

- **300+ генераторов** — Обширная база данных SSG
- **Фильтрация и поиск** — По языку, шаблонизатору, функциям
- **Таблицы сравнения** — Сравнение функций бок о бок
- **Регулярные обновления** — Данные поддерживаются сообществом

## Технологический стек

```
Astro 5 + website-core-template
├── Контент: MDX со структурированными данными
├── Стилизация: Tailwind CSS
├── i18n: Встроенная поддержка мультиязычности
├── Деплой: GitHub Pages
└── CI/CD: GitHub Actions
```

## Вклад

Проект с открытым исходным кодом. Вы можете:

- Добавлять новые SSG в базу данных
- Улучшать описания и метаданные
- Переводить контент на новые языки
- Сообщать об ошибках и предлагать новые функции

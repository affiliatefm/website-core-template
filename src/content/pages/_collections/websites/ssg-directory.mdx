## Overview

A comprehensive directory of static site generators. Compare features, filter by language, and find the perfect SSG for your next project.

## About

staticsitegenerators.net is a real-world example of what you can build with website-core-template. It showcases:

- **Multilingual content** — Same content in multiple languages with proper hreflang
- **Data-driven pages** — SSG listings generated from structured data
- **SEO optimization** — Proper meta tags, sitemap, structured data
- **Automatic deployment** — GitHub Actions → GitHub Pages

## Features

- **300+ generators** — Comprehensive database of SSGs
- **Filter & search** — By language, template engine, features
- **Comparison tables** — Side-by-side feature comparison
- **Regular updates** — Community-maintained data

## Tech Stack

```
Astro 5 + website-core-template
├── Content: MDX with structured data
├── Styling: Tailwind CSS
├── i18n: Built-in multilingual support
├── Deploy: GitHub Pages
└── CI/CD: GitHub Actions
```

## Contributing

The project is open source. You can:

- Add new SSGs to the database
- Improve descriptions and metadata
- Translate content to new languages
- Report issues and suggest features

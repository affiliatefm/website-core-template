## What It Does

Turn this:

```markdown
---
title: My Post
---

here is some text about topic. it has important info.
also there are steps: first do this, then that, finally done.
```

Into this:

```markdown
---
title: My Post
description: A comprehensive guide to topic with step-by-step instructions.
---

# My Post

Here is some text about **topic**. It contains important information.

## Steps

1. First, do this
2. Then, do that  
3. Finally, you're done

> **Note:** This guide assumes basic familiarity with the subject.
```

## Features

- **Structure enhancement** — Adds proper heading hierarchy
- **Rich blocks** — Converts patterns to callouts, tables, code blocks
- **Metadata generation** — Creates descriptions, tags from content
- **Voice preservation** — Keeps your writing style intact
- **Selective processing** — Mark files with `_enhance: true`

## Installation

```bash
npm install @affiliate.fm/astro-content-ai-enhancer
```

## Usage

Mark files for enhancement:

```yaml
---
title: My Draft
_enhance: true
---
```

Run the enhancer:

```bash
npx astro-ai-enhancer
```

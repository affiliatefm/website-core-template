## 概要

静的サイトジェネレーターの包括的なディレクトリです。機能を比較し、言語でフィルタリングして、次のプロジェクトに最適なSSGを見つけましょう。

## サイトについて

staticsitegenerators.netは、website-core-templateで構築できる実例です。以下の特徴を備えています：

- **多言語コンテンツ** — 同じ内容を複数言語で適切なhreflang付きで提供
- **データ駆動型ページ** — 構造化データから生成されるSSGリスト
- **SEO最適化** — 適切なメタタグ、サイトマップ、構造化データ
- **自動デプロイ** — GitHub Actions → GitHub Pages

## 特徴

- **300以上のジェネレーター** — SSGの包括的なデータベース
- **フィルター＆検索** — 言語、テンプレートエンジン、機能で絞り込み
- **比較表** — 機能を並べて比較
- **定期的な更新** — コミュニティによるデータ管理

## 技術スタック

```
Astro 5 + website-core-template
├── コンテンツ: 構造化データ付きMDX
├── スタイリング: Tailwind CSS
├── i18n: 組み込みの多言語対応
├── デプロイ: GitHub Pages
└── CI/CD: GitHub Actions
```

## コントリビュート

このプロジェクトはオープンソースです。あなたも以下のことができます：

- 新しいSSGをデータベースに追加
- 説明やメタデータの改善
- 新しい言語への翻訳
- 問題の報告や機能提案

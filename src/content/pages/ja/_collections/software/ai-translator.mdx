## 概要

Astroウェブサイト向けのAI翻訳ユーティリティです。`content/`ディレクトリ内のファイルからローカライズ済みコンテンツを自動生成し、シングルページ・マルチページ両方のi18nワークフローをサポートします。

## インストール

```bash
npm install @affiliate.fm/astro-content-ai-translator
npx astro-ai-translator init
```

セットアップウィザードでは以下が行われます：
- `astro.config.mjs`にインテグレーションを追加
- `.env`にOpenAI APIキーを設定

## 使い方

### 翻訳対象ファイルの指定

コンテンツファイルのfrontmatterに`_translateTo`を追加します：

```yaml
---
title: About Us
description: Learn about our company
_translateTo: [ru, de]
---
```

オプション：
- `_translateTo: [ru, de]` — 特定のロケールに翻訳
- `_translateTo: all` — 設定済みの全ロケールに翻訳
- フィールドなし — ファイルは翻訳されません

### 翻訳の実行

```bash
npx astro-ai-translator
```

翻訳開始前に以下が表示されます：
- **コスト見積もり** — 想定トークン数とUSDコスト
- **ファイルリスト** — 翻訳対象ファイル
- **確認プロンプト** — APIクレジット消費前に承認

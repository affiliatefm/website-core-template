## 概要

AIアシスタントが生のMarkdown下書きを、見やすく構造化されたページに変換します。見出し階層やコールアウトブロック、テーブルを追加し、メタデータも充実させます — あなたの文体や内容はそのままに。

## できること

このようなMarkdownを:

```markdown
---
title: My Post
---

here is some text about topic. it has important info.
also there are steps: first do this, then that, finally done.
```

このように変換します:

```markdown
---
title: My Post
description: A comprehensive guide to topic with step-by-step instructions.
---

# My Post

Here is some text about **topic**. It contains important information.

## Steps

1. First, do this
2. Then, do that  
3. Finally, you're done

> **Note:** This guide assumes basic familiarity with the subject.
```

## 特長

- **構造の強化** — 適切な見出し階層を追加
- **リッチなブロック** — パターンをコールアウトやテーブル、コードブロックに変換
- **メタデータ生成** — 内容から説明文やタグを自動作成
- **文体保持** — あなたの書き方をそのまま維持
- **選択的処理** — `_enhance: true` でファイルを指定可能

## インストール

```bash
npm install @affiliate.fm/astro-content-ai-enhancer
```

## 使い方

強化したいファイルをマークします:

```yaml
---
title: My Draft
_enhance: true
---
```

エンハンサーを実行します:

```bash
npx astro-ai-enhancer
```

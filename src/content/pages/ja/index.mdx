# Affiliate.FM コア

このテンプレートは、Astro 5 + 静的配信 + 多言語サイトを前提に最適化されています。`make bootstrap` → 設定ファイル、`make seed` → デモデータ、`make update` → 常に最新のコア。

<DataShowcase />

## 使い方

1. `npm create astro@latest -- --template affiliatefm/website-core-template`
2. `make install && make bootstrap && make seed`
3. `src/content/pages/ja/` に記事を追加
4. `src/config/site.ts` でナビゲーションとドメインを設定

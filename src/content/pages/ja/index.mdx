# Astro i18n ボイラープレート

**Astro 5** で多言語ウェブサイトを構築するための、クリーンでミニマルな基盤です。

## 特徴

- **コンテンツ主導型** — フロントマター付きMDXファイルとしてページを管理
- **自動リンク** — パス構造によって翻訳を自動的にマッチング
- **カスタムURL** — 必要に応じてロケールごとに異なるスラッグを設定可能
- **型安全** — 全体を通して完全なTypeScriptサポート
- **ミニマル** — 不要な依存関係や複雑さなし

## デモページ

- [ホーム](/) — `/ru` とペア（自動マッチ）
- [アバウト](/about) — カスタムURL: `/ru/o-nas`
- [コンタクト](/contact) — 英語のみ
- [ドキュメント](/docs/getting-started) — ネストされたパス、自動マッチ

## クイックスタート

```bash
npm install
npm run dev
```

## 仕組み

コンテンツは `src/content/pages/` に保存されています。フォルダ構造がURLを決定します:

```
src/content/pages/
├── index.mdx           → /
├── about.mdx           → /about
└── ru/
    ├── index.mdx       → /ru
    └── o-nas.mdx       → /ru/o-nas
```

同じ構造のページは自動的にリンクされます。カスタムURLはフロントマターの `permalink` と `alternates` を使用します。

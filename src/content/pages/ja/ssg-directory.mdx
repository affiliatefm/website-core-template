## 概要

staticsitegenerators.netは、website-core-templateでどんなサイトが作れるかを示す実例です。以下の特徴があります：

- **多言語対応コンテンツ** — 同じ内容を複数言語で適切なhreflang付きで提供
- **データ駆動型ページ** — 構造化データから生成されるSSGリスト
- **SEO最適化** — 適切なメタタグ、サイトマップ、構造化データ
- **自動デプロイ** — GitHub Actions → GitHub Pages

## 機能

- **300以上のジェネレーター** — SSGの充実したデータベース
- **絞り込み＆検索** — 言語、テンプレートエンジン、機能でフィルタ
- **比較表** — 機能を並べて比較
- **定期的な更新** — コミュニティによるデータ管理

## 技術スタック

```
Astro 5 + website-core-template
├── コンテンツ: 構造化データ付きMDX
├── スタイリング: Tailwind CSS
├── i18n: 組み込みの多言語サポート
├── デプロイ: GitHub Pages
└── CI/CD: GitHub Actions
```

## プロジェクト構成

```
src/
├── content/pages/     # ✏️ コンテンツ（MDXファイル）
│   ├── index.mdx      #    トップページ
│   ├── about.mdx      #    別ページ
│   └── ru/            #    ロシア語コンテンツ
│       └── index.mdx
├── data/              # ✏️ 設定ファイル
│   ├── site.ts        #    サイトURL、デフォルト言語
│   └── ui.ts          #    UI翻訳
├── config/            #    実行時設定（自動生成）
├── i18n/              #    URLヘルパー
├── integrations/      #    ビルドチェック
├── layouts/           #    ページレイアウト
└── pages/             #    Astroルーター
```

**あなたのファイル**（✏️） — 自由に編集可能、アップデートで上書きされません  
**コアファイル** — `make update` で更新されます

## 設定

### 1. サイトの設定

`src/data/site.ts` を編集します:

```typescript
export const siteUrl = "https://your-domain.com";
export const defaultLanguage = "en";
```

### 2. UI文字列の追加

`src/data/ui.ts` を編集します:

```typescript
export const uiStrings = {
  en: { siteName: "My Site" },
  ru: { siteName: "Мой сайт" },
} as const;
```

## コンテンツの追加

`src/content/pages/` にMDXファイルを作成します:

```yaml
---
title: "ページタイトル"
description: "SEO用のメタディスクリプション"
---

# ここにコンテンツを記述

Markdownで記述でき、MDXコンポーネントもフルサポートされています。
```

### フロントマターのオプション

| フィールド | 型 | 説明 |
|-------|------|-------------|
| title | string | ページタイトル（必須） |
| description | string | メタディスクリプション |
| permalink | string | カスタムURLスラッグ |
| alternates | object | 言語 → スラッグのマッピング |
| updatedAt | date | サイトマップ用 |
| draft | boolean | ビルドから除外 |

## 多言語対応の設定

### 同じスラッグ（自動）

```
pages/about.mdx      → /about/
pages/ru/about.mdx   → /ru/about/
```

Hreflangリンクは自動生成されます。

### 異なるスラッグ

```yaml
# pages/about.mdx
---
title: About
alternates:
  ru: o-nas
---

# pages/ru/o-nas.mdx
---
title: О нас
permalink: o-nas
alternates:
  en: about
---
```

### 外部のオルタネート

```yaml
---
title: About
alternates:
  ja: https://ja.example.com/about/
---
```

## ツールによる拡張

このテンプレートは他の Affiliate.FM ツールとシームレスに連携します:

<RatingTable products={getProductsByRank().slice(1, 3)} />

### AI翻訳

自動翻訳をワークフローに追加できます:

```bash
npm install @affiliate.fm/astro-content-ai-translator
npx astro-ai-translator init
```

翻訳対象ファイルをマークします:

```yaml
---
title: About Us
_translateTo: all
---
```

### インテグレーションの追加

```bash
npx astro add tailwind
npx astro add react
```

## アップデート

このテンプレートは安全にアップデートできるよう設計されています:

```bash
# 変更点を確認
make update-check

# アップデートを適用
make update
```

**保持されるもの:** `src/content/`, `src/data/`, `public/`  
**更新されるもの:** それ以外すべて

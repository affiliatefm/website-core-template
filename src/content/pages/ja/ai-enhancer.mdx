## できること

このようなMarkdownを:

```markdown
---
title: My Post
---

here is some text about topic. it has important info.
also there are steps: first do this, then that, finally done.
```

このように変換します:

```markdown
---
title: My Post
description: A comprehensive guide to topic with step-by-step instructions.
---

# My Post

Here is some text about **topic**. It contains important information.

## Steps

1. First, do this
2. Then, do that  
3. Finally, you're done

> **Note:** This guide assumes basic familiarity with the subject.
```

## 主な特徴

- **構造強化** — 適切な見出し階層を自動で追加
- **リッチブロック** — パターンをコールアウト、テーブル、コードブロックへ変換
- **メタデータ生成** — 内容から説明文やタグを自動生成
- **文体維持** — あなたの書き方をそのまま保持
- **選択的処理** — `_enhance: true` でファイルを指定

## インストール

```bash
npm install @affiliate.fm/astro-content-ai-enhancer
```

## 使い方

強化したいファイルにマークを付けます:

```yaml
---
title: My Draft
_enhance: true
---
```

エンハンサーを実行:

```bash
npx astro-ai-enhancer
```

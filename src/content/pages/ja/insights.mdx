# データとインサイト

`src/data/` に置いた JSON を Astro/MDX から直接読み込み、翻訳済みコンテンツと同じテンプレートで表示できます。

1. `make seed` でデモデータを生成
2. `@/lib/data` のヘルパーで取り込み
3. `<DataShowcase />` のようなコンポーネントでレンダリング

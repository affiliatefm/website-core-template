# About This Template

This boilerplate demonstrates how to build a **content-driven multilingual site** with Astro 5's built-in i18n features.

## Architecture

### Configuration (`src/config/site.ts`)

**Single source of truth** for all site settings:

- Supported locales and default locale
- Site URL and metadata
- UI translations and navigation links

### Content (`src/content/pages/`)

MDX files organized by locale. Default locale files live at the root, other locales in subfolders.

### Routing (`src/pages/`)

Single universal route handles all pages:

- `[...slug].astro` — All locales, all paths

## Custom URLs

This page demonstrates **custom URL mapping**:

- English: `/about`
- Russian: `/ru/o-nas`

Achieved with `alternates` in frontmatter on both pages, plus `permalink` on the Russian version.

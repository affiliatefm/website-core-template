# Website Core Template

Reusable, updatable static site generator for multilingual sites. Part of the [Affiliate.FM](https://affiliate.fm) open-source ecosystem — tools for affiliate and digital marketing.

## Features

- **i18n routing** — folder-based, zero config
- **Auto-sitemap** — with hreflang links
- **Updatable core** — `make update` syncs core, preserves your content
- **Data-ready** — import data into MDX pages

## Quick Start

```bash
npm create astro@latest -- --template affiliatefm/website-core-template
make dev
```

Edit `src/data/site.ts` → add content to `src/content/pages/` → done.

## Docs

### Adding Languages

Create `src/content/pages/ru/index.mdx` → available at `/ru/`

### Frontmatter

- `title` — page title (required)
- `description` — meta description
- `permalink` — custom URL slug
- `alternates` — link translations with different slugs
- `draft: true` — exclude from build

### Commands

| Command | Description |
|---------|-------------|
| `make dev` | Dev server |
| `make build` | Production build |
| `make update` | Update core |

# Affiliate.FM

Біз тәуелсіз медиа және open-source құралдарды жасаймыз. Бұл репо — Astro 5 негізіндегі статикалық ядро.

- Жергілікті контентті `src/content/pages/ru-kz/` ішіне салыңыз.
- Makefile командалары (мысалы `make dev`, `make update`) тәжірибесіз қолданушыларға да түсінікті.
- `affiliatefm.core.json` файлы core vs. user layer шекарасын белгілейді.

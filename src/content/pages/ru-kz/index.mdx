# Astro i18n Бойлерплейт (Қазақстан)

Көптілді сайттар құруға арналған таза негіз **Astro 5**.

## Мүмкіндіктер

- **Content-driven** — MDX файлдар түріндегі беттер
- **Автосілтеме** — Аудармалар құрылым бойынша байланысады
- **Реттелетін URL** — Әр тіл үшін әртүрлі slug
- **Типизация** — TypeScript қолдауы
- **Минимализм** — Артық тәуелділіктерсіз

## Демо беттер

- [Басты бет](/ru-KZ) — Қазақстан нұсқасы
- [Біз туралы](/ru-KZ/biz-turaly) — Қазақша URL

## Тез бастау

```bash
npm install
npm run dev
```

## Бұл қалай жұмыс істейді

Мазмұн `src/content/pages/` ішінде. Қалта құрылымы URL анықтайды:

```
src/content/pages/
├── index.mdx           → /
├── ru/
│   └── index.mdx       → /ru
└── ru-KZ/
    └── index.mdx       → /ru-KZ
```
